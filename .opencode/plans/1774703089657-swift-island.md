### Middle Ellipsis Function
```python
def truncate_middle(text: str, max_width: int) -> str:
    if max_width <= 0:
        return ""
    if len(text) <= max_width:
        return text
    if max_width == 1:
        return "\u22ee"
    keep = max_width - 1
    left = keep // 2
    right = keep - left
    return f"{text[:left]}\u22ee{text[-right:]}" if right else f"{text[:left]}\u22ee"
```
For pixel-based libraries (Qt, wx, etc.), use font metrics to calculate `max_width` in characters, then call `truncate_middle`. Do NOT rely on native `elideMiddle`/`NSLineBreakByTruncatingMiddle` ŌĆö always produce `Ōŗ«` (U+22EE).
