Vexy Lines is a desktop Qt 5 C++ app that you can use to open a photo, illustration or generated image, segment that source image into masks and then apply different algorithmic fills to the segments of the image. The app can export vector graphics documents (SVG, PDF) that can simulate halftoning, engraving and other sophisticated effects, both monochrome and multicolor. Vexy Lines saves its native documents save as `.lines` files. The documents have groups (each group can have its own source image) and layers (each layer can have its own mask, and multiple fills). The app also supports 3D mesh warping and overlap control for woven effects. 

- _private/vlum-tldr.md contains the functional TLDR of the Vexy Lines app 
- _private/vexy-lines-src-tldr.txt contains the TLDR of the Vexy Lines app source code 
- _private/vexy-lines-src-readonly/ contains a readonly snapshot of the Vexy Lines app source code 
- _private/mcp/ contains the source code and Python example of the external API of the Vexy Lines desktop app. The API is compatible with the MCP (Model Context Protocol). 
- The main folder is for `vexy-lines-utils`, a simple Python tool that works on macOS and drives the Vexy Lines app through OS scripting, and allows exporting `.lines` documents to PDF or SVG.
